# Real Estate Data Analytics & Price Prediction

## 📊 Project Overview
End-to-end data analytics and ML project to predict property prices in Gurgaon.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly

## 🚀 Features
- Data cleaning & preprocessing
- EDA & visualization
- Price prediction model
- Recommendation system
- Interactive Streamlit app

- app/
- data/
